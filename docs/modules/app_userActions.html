    Module app/userActions

    Utilities for managing application users (server-actions).

    This module exposes server-side actions to create, update, list and delete users. Each function expects a FormData payload and performs authentication checks using auth0.

    Remarks

    These functions are intended to be used as server actions in a Next.js app and will redirect to '/editor' when no valid session is present.
